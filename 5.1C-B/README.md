# Task 5.1C (B) - iStream YouTube App

## Project Overview

A YouTube streaming app, with users able to specify YouTube URLs for streaming, and add videos to a local playlist. 

### Tech Stack
1. **Java** - Core logic
2. **XML (Material Design 3)** - UI layout
3. **Android View System** - UI paradigm
4. **Android Studio** - IDE

## Key Features

- Allows for multiple users
- Password authentication
- User playlists stored privately using Room database 

## Installation

Open this folder in Android Studio and run on a suitable emulator:  
**Minimum OS:** API 24 (Nougat)  
**Target OS:** API 36 (Baklava)
