# Task 7.1P - Lost & Found App

## Project Overview

An app to connect lost items to their owners, with user-defined adverts displayed in a list. Each lost or found item is listed with fields like name, location, and image. 

### Tech Stack

1. **Kotlin** - Logic and UI
2. **Jetpack Compose** - UI framework
3. **Room** - Database ORM
4. **Android Studio** - IDE

## Key Features

- User-defined lost and found listings
- Item list sorted by date
- Listings are removable

## Implementation

- MVVM (Model-View-ViewModel) architecture
- NavHost with Composables provides single-activity app
- Local persistent data using Room

## Installation

Open this folder in Android Studio and run on a suitable emulator:  
**Minimum OS:** API 24 (Nougat)  
**Target OS:** API 36 (Baklava)
