# Task 4.1P - Event Planner App

## Project Overview

An event planner app, listing user-defined events by date. Each event has a title, date, and time, along with optional location and category. 

### Tech Stack

1. **Kotlin** - Logic and UI
2. **Jetpack Compose** - UI framework
3. **Room** - Database ORM
4. **Android Studio** - IDE

## Key Features

- User-defined events
- Event list sorted by date
- Events are modifiable and removable

## Implementation

- MVVM (Model-View-ViewModel) architecture
- NavHost with Composables provides single-activity app
- Local persistent data using Room

## Installation

Open this folder in Android Studio and run on a suitable emulator:  
**Minimum OS:** API 24 (Nougat)  
**Target OS:** API 36 (Baklava)
