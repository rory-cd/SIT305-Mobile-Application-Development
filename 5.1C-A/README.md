# Task 5.1C (A) - Sports News App

## Project Overview

A sports news feed, allowing users to browse and bookmark news articles.

### Tech Stack
1. **Java** - Core logic
2. **XML (Material Design 3)** - UI layout
3. **Android View System** - UI paradigm
4. **Android Studio** - IDE

## Key Features

- Article filtering provides quick text-based search
- Related articles for ease of navigation
- Bookmarks menu shows all bookmarked articles in one location

## Installation

Open this folder in Android Studio and run on a suitable emulator:  
**Minimum OS:** API 24 (Nougat)  
**Target OS:** API 36 (Baklava)
