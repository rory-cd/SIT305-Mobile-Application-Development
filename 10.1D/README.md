# Task 6.1P - AI-Powered Learning App

## Project Overview

A quiz-based personalised learning app, using an LLM to generate quizzes based on user-selected topics. After each quiz, a personalised lesson plan is produced.

### Tech Stack

1. **Kotlin** - Logic and UI
2. **Jetpack Compose** - UI framework
3. **Room** - Database ORM
4. **Android Studio** - IDE
5. **Flask** - AI web server (Not included)

## Key Features

- AI-generated quizzes
- Personalised responses with learning plan recommendations
- User accounts for multiple users

## Implementation

- MVVM (Model-View-ViewModel) architecture
- NavHost with Composables provides single-activity app
- Local persistent data using Room

## Installation

Open this folder in Android Studio and run on a suitable emulator:  
**Minimum OS:** API 24 (Nougat)  
**Target OS:** API 36 (Baklava)
